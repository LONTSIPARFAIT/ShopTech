# Tableaux de Bord

## 1. Dashboard Admin
Une interface centralisée pour piloter l'activité.
- **KPIs** : Revenus totaux, nombre de commandes, produits en stock, nouveaux clients.
- **Ventes** : Liste des commandes récentes avec accès rapide aux détails.
- **Gestion** : Accès rapide à la création de produits.

## 2. Espace Client
Permet aux clients de suivre leurs interactions avec la boutique.
- **Historique** : Liste exhaustive de toutes les commandes passées.
- **Détails** : Récapitulatif précis d'une commande (articles, prix d'achat, statut de livraison).
